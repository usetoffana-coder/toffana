📊 Configurações de Tracking

📘 Facebook Pixel

Encontre seu Pixel ID em: Eventos do Facebook > Origens de Dados

🧾 Eventos Rastreados:

  • PageView: Visualização de página
  • ViewContent: Visualização de produto
  • Contact: Clique no WhatsApp

🏷️ Google Tag Manager

Formato: GTM-XXXXXXX (encontre em: Google Tag Manager > Workspace)

🧾 DataLayer Events:

  • page_view: Visualização de página
  • view_item: Visualização de produto
  • whatsapp_contact: Contato via WhatsApp

📈 Google Analytics 4

Formato: G-XXXXXXXXXX (encontre em: Admin > Propriedade > Fluxos de Dados)

🧾 Eventos GA4:

  • page_view: Visualização automática
  • view_item: Produto visualizado
  • contact: Contato WhatsApp

⚙️ Configurações Avançadas

Registra quando usuário rola 25%, 50%, 75% e 100% da página
Envia evento quando usuário passa mais de 30s, 1min e 3min
